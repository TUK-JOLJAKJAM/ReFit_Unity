# ReFit
- 게임화 스마트 재활 시스템
- Gamification Smart Rehabilitation System

# 게임 파트
- 목적 : 지루한 반복 재활 동작을 게임의 조작으로 바꾸어 재활 동작을 재미있게, 더 효과적으로 할 수 있게 유도
- 구성 : 각 부위 별 게임, 스트레칭 파트, 재화 사용

# 부위 별 게임
## 1. 장작 패기 - 어께
- 개요 : 정해진 장작을 시간 내에 정확히 쪼개는 게임
- 동작 : 팔을 앞으로 들어 올려 뒤로 넘기는 신전 운동 

## 2. 마을 지키기 - 허리
- 개요 : 날아오는 적을 추적하여 격추하는 게임
- 동작 : 허리를 좌우로 비트는 운동
